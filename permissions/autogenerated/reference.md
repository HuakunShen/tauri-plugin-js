## Default Permission

Default permissions for the plugin

#### This default permission set includes the following:

- `allow-spawn`
- `allow-kill`
- `allow-kill-all`
- `allow-restart`
- `allow-list-processes`
- `allow-get-status`
- `allow-write-stdin`
- `allow-detect-runtimes`
- `allow-set-runtime-path`
- `allow-get-runtime-paths`
